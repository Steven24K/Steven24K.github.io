Laatste nieuws GitHub als Content Management Systeem(CMS)
Deze website is gemaakt met Typescript en React, verder heb ik gberuik gemaakt van Bootstrap 4 voor de CSS en wordt hehost vanuit een public repository via GitHub pages. De reden dat ik voor Bootstrap heb gekozen is omdat ik zelf meer een back-end developper ben en met de hulp van Bootstrap kan je vrij snel een mooi opgemaakte website neerzetten. Het idee achter voor de keuze voor React zit hem in de React-Router, dit zorgt ervoor dat je componenten die eenmalig hebt gemaakt makkelijk kunt aanroepen vanuit de navigatiebalk.
Verder over GitHub pages, ik heb de website zo ontworpen dat ik makkelijk items kan toevoegen door simpel weg een bestand aan te maken of te bewerken en vervolgens dit bestand te pushen naar GitHub. Op deze mannier heb ik geen ingewikkeld en complex Content Management Systeem nodig, geen dure hosting pakketten(GitHub pages is gratis), geen FTP explorers, maar alleen een gratis GitHub account.
